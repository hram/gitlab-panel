# GitLab Panel

Панель управления для мониторинга проектов GitLab, релизов, веток и стадий разработки.

## Описание

Веб-приложение на **FastAPI** для управления проектами GitLab и отслеживания жизненного цикла релизов. Приложение предоставляет интерфейс для:

- Управления проектами (добавление, просмотр, удаление)
- Просмотра веток разработки
- Отслеживания стадий релизов
- Истории изменений стадий релизов

## Архитектура

Проект использует чистую архитектуру с разделением на слои:

- **`app/api/`** — маршруты и обработчики запросов
- **`app/application/`** — бизнес-логика (сервисы)
- **`app/domain/`** — доменные модели и репозитории
- **`app/infrastructure/`** — работа с базой данных
- **`templates/`** — HTML-шаблоны Jinja2
- **`static/`** — статические файлы (CSS)

## Технологии

- **FastAPI** — веб-фреймворк
- **Jinja2** — шаблонизатор
- **SQLite** — база данных
- **python-gitlab** — клиент для работы с GitLab API
- **Uvicorn** — ASGI-сервер

## Установка и запуск

### 1. Активация виртуального окружения

```bash
source venv/bin/activate
```

### 2. Запуск сервера разработки

```bash
uvicorn app.main:app --reload
```

После запуска приложение будет доступно по адресу: **http://127.0.0.1:8000**

## Структура базы данных

Приложение использует SQLite (`projects.db`) со следующими таблицами:

- **projects** — проекты GitLab
- **releases** — релизы проектов
- **stages** — стадии разработки
- **release_stage_history** — история изменений стадий релизов

## API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница |
| GET | `/branches` | Страница веток |
| GET | `/branches/{project_id}` | Ветки конкретного проекта |
| GET | `/projects` | Список проектов |
| POST | `/projects/create` | Создание проекта |
| POST | `/projects/delete/{project_id}` | Удаление проекта |
| GET | `/projects/{project_id}/releases` | Релизы проекта |
| GET | `/projects/{project_id}/stages` | Стадии проекта |
| GET | `/api/branches` | API: список веток |
| GET | `/api/branches/{project_id}` | API: ветки проекта |
